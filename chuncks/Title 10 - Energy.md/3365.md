(ii)If a licensee has previously applied a Code Case and a later version of the Code Case is incorporated by reference in paragraph (a) of this section, the licensee may continue to apply, to the end of the current 120-month interval, the previous version of the Code Case, as authorized, or may apply the later version of the Code Case, including any NRC-specified conditions placed on its use.
