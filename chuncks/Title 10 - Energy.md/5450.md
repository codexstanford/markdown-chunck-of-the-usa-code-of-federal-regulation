###### Lock,Lock,reading,Locked

in the case of vaults or vault type rooms means a three-position, manipulation resistant, dial type, built-in combination lock or combination padlock and in the case of fences, walls, and buildings means an integral door lock or padlock which provides protection equivalent to a six-tumbler cylinder lock.in the case of a vault or vault type room also means any manipulation resistant, electromechanical device which provides the same function as a built-in combination lock or combination padlock, which can be operated remotely or by theor insertion of information, which can be uniquely characterized, and which allows operation of the device.means protected by an operable lock.
