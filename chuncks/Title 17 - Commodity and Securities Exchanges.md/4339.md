
# Title 17 - Commodity and Securities Exchanges
## Chapter I - Commodity Futures Trading Commission
### PART 49 - SWAP DATA REPOSITORIES
#### ยง 49.10 Acceptance and validation of data.
##### Data validation acceptance message.

(1)A swap data repository shall validate each SDR data report submitted to the swap data repository and notify the reporting counterparty, swap execution facility, designated contract market, or third-party service provider submitting the report whether the report satisfied the data validation procedures of the swap data repository as soon as technologically practicable after accepting the SDR data report.
