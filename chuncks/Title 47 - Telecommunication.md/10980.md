(d)(1) Decoders must support Code Space C0, G0, C1, and G1 in their entirety.

(i) Transparent space ().

(ii) Non-breaking transparent space ().

(iii) Solid block ( ).

(iv) Trademark symbol ().

(v) Latin-1 characters &#x160;, &#x152;, &#x161;, &#x153;, &#x178;.

(3) The substitutions in Table 2 are to be made if a decoder does not support the remaining G2 characters.

(4) Support for code spaces C2, C3, and G3 is optional. All unsupported graphic symbols in the G3 code space are to be substituted with the G0 underscore character (_), char code 0 x 5F.
