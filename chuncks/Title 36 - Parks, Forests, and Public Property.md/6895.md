
# Title 36 - Parks, Forests, and Public Property
## Chapter XI - Architectural and Transportation Barriers Compliance Board
### PART 1194 - INFORMATION AND COMMUNICATION TECHNOLOGY STANDARDS AND GUIDELINES
#### Appendix C to Part 1194 - Functional Performance Criteria and Technical Requirements
##### Content Creation or Editing.,see

504.2Authoring tools shall provide a mode of operation to create or edit content that conforms to Level A and Level AA Success Criteria and Conformance Requirements in WCAG 2.0 (incorporated by reference,702.10.1) for all supported features and, as applicable, to file formats supported by the authoring tool. Authoring tools shall permit authors the option of overriding information required for accessibility.

EXCEPTION: Authoring tools shall not be required to conform to 504.2 when used to directly edit plain text source code.
