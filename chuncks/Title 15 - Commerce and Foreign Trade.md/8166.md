###### Subpart C - Requirements for Certified Distributors and Value Added Distributors of NOAA ENC Products

######## Encryption.

(iii)The NOAA ENC files may be encrypted by CED, providing that the encryption/decryption process does not result in any information loss and that CED makes the decryption software available to the end user as part of the redistribution service. Decrypted files must have the same CRC checksum value as the original files. CED shall make the encryption/decryption software and documentation available to NOAA for testing.
