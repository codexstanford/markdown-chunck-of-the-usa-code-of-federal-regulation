
# Title 46 - Shipping
## Chapter I - Coast Guard, Department of Homeland Security
### Subchapter O - Certain Bulk Dangerous Cargoes
#### PART 154-SAFETY STANDARDS FOR SELF-PROPELLED VESSELS CARRYING BULK LIQUEFIED GASES
##### Subpart C-Design, Construction and Equipment
###### Instrumentation
####### ยง 154.1365 Audible and visual alarms.

(a) Each audible alarm must have an arrangement that allows it to be turned off after sounding. For remote group alarms this arrangement must not interrupt the alarm's actuation by other faults.

(b) Each visual alarm must be one that can be turned off only after the fault that actuated it is corrected.

(c) Each visual alarm must be marked to show the type and, except for remote group alarms, the location of each fault that actuates it.

(d) Each vessel must have means for testing each alarm.
