####### Data elements for follow-up youth.

(e)The State agency must report the data elements described in paragraphs (g)(34) through (g)(58) of this section for each youth in the follow-up population defined in section 1356.81(c) of this part or alternatively, for each youth selected in accordance with the sampling procedures in section 1356.84 of this part.
