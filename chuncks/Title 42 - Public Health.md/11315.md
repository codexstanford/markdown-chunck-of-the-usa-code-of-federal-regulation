
# Title 42 - Public Health

### Subchapter B - Medicare Program

##### Subpart D - Payment for Durable Medical Equipment and Prosthetic and Orthotic Devices
###### ยง 414.210 General payment rules.
####### Adjusted payment amounts for accessories used with different types of base equipment.

(5)In situations where a HCPCS code that describes an item used with different types of base equipment is included in more than one product category in a CBA under competitive bidding, a weighted average of the single payment amounts for the code is computed for each CBA based on the total number of allowed services for the item on a national basis for the code from each product category prior to applying the payment adjustment methodologies in this section.
