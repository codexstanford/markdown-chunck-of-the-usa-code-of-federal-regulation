####### Test conditions,Test species,Selection.,Daphnia magna,D. pulex,

(d)-(1)-(i)(A) The cladocerans,orare the test species to be used in this test. Either species may be used for testing of a particular chemical. The species identity of the test organisms should be verified using appropriate systematic keys. First instar daphnids, &#x2264;24 hours old, are to be used to start the test.

(B) Daphnids to be used in acute toxicity tests should be cultured at the test facility. Records should be kept regarding the source of the initial stock and culturing techniques. All organisms used for a particular test shall have originated from the same culture population.
