
# Title 26 - Internal Revenue

####### ยง 301.6081-2 Automatic extension of time for filing an information return with respect to certain foreign trusts.
######## In general.

(a)A trust required to file a return on Form 3520-A, "Annual Information Return of Foreign Trust with a U.S. Owner," will be allowed an automatic 6-month extension of time to file the return after the date prescribed for filing the return if the trust files an application under this section in accordance with paragraph (b) of this section.
