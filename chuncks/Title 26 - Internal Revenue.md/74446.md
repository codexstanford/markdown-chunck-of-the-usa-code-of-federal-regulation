######## Loss year return by the fiduciary.

(3)If the institution is a member of a loss year group, and either the common parent does not file a loss year return or the fiduciary does not accept the loss year return filed by the common parent, the fiduciary may file a loss year return with respect to the loss year group. A loss year return can only be filed by the fiduciary in conjunction with the filing of a claim for refund under paragraph (e)(1).The return must be based on all information pertaining to the institution and all information pertaining to other members to which the fiduciary has reasonable access. Any return filed by the fiduciary under this paragraph (e)(3) must contain the title "Loss year return under section 6402(i) of the Code" at the top of the first page of the return, and the following must be attached to the return:

(i) The name and employer identification number of the institution that is a member of the loss year group;

(ii) The name of the fiduciary;

(iii) A representation that the institution is an insolvent financial institution as defined in paragraph (b)(4) of this section; and

(iv) A representation that the fiduciary has satisfied the requirements set forth in paragraphs (d)(2)(i) and (ii) of this section.
