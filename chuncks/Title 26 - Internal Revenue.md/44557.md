
# Title 26 - Internal Revenue

###### ยง 1.1061-2 Applicable partnership interests and applicable trades or businesses.
####### Example 2: Contribution of an API to a partnership.

(ii)Individuals A, B, and C each directly hold APIs in PRS, a partnership. A and B form a new partnership, GP, and contribute their APIs in PRS to GP. Following the contribution, each of A and B holds an Indirect API because each of A and B now indirectly holds an API in PRS through GP, a Passthrough Entity. Each of A's and B's interests in GP is a Passthrough Interest because each of A's and B's interest in GP represents an Indirect API.
