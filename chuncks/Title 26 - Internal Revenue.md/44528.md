
# Title 26 - Internal Revenue

### Subchapter A - Income Tax
#### PART 1 - INCOME TAXES
##### Special Rules
###### ยง 1.1061-1 Section 1061 definitions.
####### Distributed API Property

means property distributed by a Passthrough Entity to an API Holder with respect to an API if the holding period, as determined under sections 735 and 1223, in the API Holder's hands is three years or less at the time of disposition of the property by the API Holder.
