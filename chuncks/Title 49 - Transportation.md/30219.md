
# Title 49 - Transportation
## Subtitle B - Other Regulations Relating to Transportation
### Chapter XII - Transportation Security Administration, Department of Homeland Security
#### Subchapter C - Civil Aviation Security
##### PART 1540 - CIVIL AVIATION SECURITY: GENERAL RULES
###### Subpart A - General
####### ยง 1540.5 Terms used in this subchapter.
######## Standard security program

means a security program issued by TSA that serves as a baseline for a particular type of operator. If TSA has issued a standard security program for a particular type of operator, unless otherwise authorized by TSA, each operator's security program consists of the standard security program together with any amendments and alternative procedures approved or accepted by TSA.
