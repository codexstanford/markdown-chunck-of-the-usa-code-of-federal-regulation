### PART 1005-ELECTRONIC FUND TRANSFERS (REGULATION E)
#### Supplement I to Part 1005-Official Interpretations
##### Machine-readable text.

3.Section 1005.18(b)(6)(i)(B) requires that electronic disclosures must be provided using machine-readable text that is accessible via both Web browsers (or mobile applications, as applicable) and screen readers. A disclosure would not be deemed to comply with this requirement if it was not provided in a form that can be read automatically by internet search engines or other computer systems.
