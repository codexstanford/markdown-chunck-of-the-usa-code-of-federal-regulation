
# Title 16 - Commercial Practices

###### ยง 1610.4 Requirements for classifying textiles.
####### Class 1, Normal Flammability.

(a)Class 1 textiles exhibit normal flammability and are acceptable for use in clothing. This class shall include textiles which meet the minimum requirements set forth in paragraph (a)(1) or paragraph (a)(2) of this section.
